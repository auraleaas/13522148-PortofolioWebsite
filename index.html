WikiScrapper - Web-based Wiki Race Game

Developed a dynamic web interface using Next.js, Tailwind CSS, and TypeScript, allowing users to input URLs and track their progress in the WikiRace, a game where players navigate from one Wikipedia page to another using internal links. Implemented Breadth-First Search (BFS) and Iterative Deepening Search (IDS) algorithms to find paths between Wikipedia pages, and utilized web scraping to gather necessary data, creating an intuitive GUI to visualize the search process and display results.

Github

LENS - Content-Based Image Retrieval (CBIR) Web

A web application designed to enhance image search and retrieval based on visual content rather than metadata or keywords. Users can upload an image, and LENS analyzes features like color, texture, and shape using advanced algorithms. These features are compared with a large database to find visually similar images. The result is a ranked list of images that closely match the uploaded image, providing an efficient and accurate tool for image retrieval.
